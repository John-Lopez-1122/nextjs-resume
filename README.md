# John Lopez — Resume Page

A single-page resume built with Next.js and styled using Tailwind CSS.

## How to Run Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Resume content in `src/app/page.tsx` (name, contact, education, experience, projects, skills)
- Semantic HTML elements (`<header>`, `<main>`, `<section>`, `<footer>`)
- All styling done with Tailwind CSS utility classes
- Clean, readable layout at desktop viewport
- No console errors

## Known Issues / Limitations

- Layout is optimized for desktop
